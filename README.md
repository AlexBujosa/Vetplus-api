# Vetplus Api Backend

The Vetplus backend was made with [Nest.js](https://github.com/nestjs/nest) framework

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Setting up Prisma

```bash
# generate types for @prisma/client
$ npx prisma generate

# push changes to prisma db
$ npx prisma db push
```

## Graphql Playground

```bash
# run app in development mode
$ localhost:3000/graphql
```
